Little Compton Stone Wall Stewards

About

This application showcases data collected by volunteers as part of an effort to document the stone walls of Little Compton, Rhode Island.

Each point on the map represents an individual observation. When selected, more information is displayed.

If you're interested in volunteering, please follow this link. For instructions on collecting data, please see this instructional document. For instructions on how to use the mobile data collection app, please watch this video.
